# Titanic Analysis

Data analysis project based on the Titanic dataset using Python, Pandas, and NumPy.

## Project Goals

- Load and explore the Titanic dataset
- Clean and preprocess data
- Analyze passenger survival patterns
- Generate statistical insights
- Visualize important findings

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib

## Project Structure

```text
titanic-analysis/
│
├── data/
├── data_processing/
├── main.py
├── requirements.txt
└── README.md
```

## Installation

```bash
git clone https://github.com/Farhad0626/titanic-analysis.git
cd titanic-analysis

python -m venv .venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt
```

## Run

```bash
python run.py
```

## Dataset

Titanic dataset from Kaggle.
